## Project Context

**Project Name:** Maybridge Publishing Landing Page Enhancement

**Objective:** Enhance the existing Next.js website to showcase 110 educational books categorized into Primary, Junior High, and Senior High levels. Each book will be displayed as an interactive card with links to Amazon purchases and individual detail pages. Additional sections include About Us, Contact, Hero, Testimonials, Newsletter, and Footer.

**Technology Stack:** Next.js, Prismic CMS, TypeScript, Tailwind CSS, Framer Motion, x/ui library (or shadcn/ui alternative).

**Timeline:** 2-3 weeks sprint.

**Business Goals:** Increase book sales by 20-30%, improve brand awareness, enhance user engagement, reduce bounce rates by 15%.

**Target Audience:** Parents, educators, students, and potential partners interested in educational materials.

**Key Features:**
- Book showcase with 110 books in categories
- Individual book detail pages with galleries, reviews, editor notes
- Contact form with email integration
- Theme toggle (light/dark)
- SEO optimization and structured data
- Mobile-responsive design

**Current Status:** PRD completed, repo assessment done, doability confirmed. Ready for implementation readiness check.
